Код бичих зөвлөмж
=================

Энэхүү төслийг програмчлал хэл, технологи, фрэймворкуудад ашиглагдаж байгаа код бичих зөвлөмжүүдийг /coding conventions/
эх хэл рүүгээ орчуулан, ашиглах, түгээн дэлгэрүүлэх зорилгоор энэ төслийг үүсгэлээ.
